Default Parameters nədir? - Default Parameters JavaScript-də funksiyalara parametr verilmədikdə avtomatik istifadə olunan dəyərlərdir. - Başqa sözlə, əgər funksiya çağırılarkən bəzi parametrlər ötürülməzsə, onlara əvvəlcədən təyin edilmiş dəyər istifadə olunur. - Bu ES6 ilə gəldi və əvvəlki üsullara (param || default) nisbətən daha sadə və oxunaqlıdır.
